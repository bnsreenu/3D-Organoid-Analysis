# 3D Organoid Analysis with Python

This repository accompanies my YouTube series on **3D organoid imaging analysis**, using Python, machine learning, and modern data science tools.

**YouTube Playlist:** [3D Organoid Analysis with Python](https://www.youtube.com/playlist?list=PLZsOBAyNTZwbxUC3El8ucNvOzCSIG-ZCb)

The series walks you through the complete analytical pipeline, from raw microscopy images to actionable biological insights, using real published data.
